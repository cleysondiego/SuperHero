# Super Heroes

## Executar o projeto

### Clonar o repositório
```
$ git clone git@github.com:cleysondiego/SuperHero.git
```

### Instalar as dependências

```
$ yarn install
```

### Criar o arquivo .env

Há um exemplo de quais variáveis devem ser criadas no arquivo: `.env.example`

### Executar no emulador android

```
$ yarn android
```
